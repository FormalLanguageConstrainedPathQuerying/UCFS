# UCFSBenchmarks

## Prerequisites

```text
(1) Gradle (version >= 7.2)
(2) Antlr V4
(3) Jflex
```
## Generate files
Run `/scripts/generate_all.sh`
## Run benchmarks
Set data-set folder and parser mode in script `/scripts/run_bench.sh` and run it

## Logging

Logs are stored in `logs`

Results are stored in  `build/reports/benchmarks`
